# СтройМаркет - Интернет-магазин строительных материалов

Интернет-магазин строительных материалов и инструментов с удобным каталогом и системой фильтрации. Проект создан для демонстрации навыков разработки интернет-магазинов для специфических ниш.

## Описание проекта

Этот проект представляет собой полнофункциональный интернет-магазин строительных товаров. Сайт включает каталог товаров с фильтрацией по категориям, систему поиска, корзину покупок и информационные разделы о компании.

## Основные возможности

- Каталог товаров с фильтрацией по категориям (материалы, инструменты, отделка)
- Система поиска товаров по названию и описанию
- Корзина покупок с подсчетом количества товаров
- Адаптивный дизайн для всех устройств
- Современный интерфейс с оранжевой цветовой схемой
- Информационные секции о компании и преимуществах

## Технологии

Проект создан с использованием чистого HTML, CSS и JavaScript без фреймворков для максимальной производительности и простоты поддержки.

- HTML5 для семантической разметки
- CSS3 для стилизации с использованием Grid и Flexbox
- Vanilla JavaScript для интерактивности
- Google Fonts для типографики

## Запуск проекта

Проект не требует установки зависимостей. Просто откройте файл `index.html` в браузере.

Для разработки можно использовать локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

Затем откройте браузер и перейдите по адресу `http://localhost:8000`

## Особенности реализации

### Дизайн

Интерфейс выполнен в оранжевой цветовой гамме, что ассоциируется со строительством и энергией. Использованы крупные карточки товаров и четкая типографика для удобного просмотра каталога.

### Категории товаров

Сайт включает шесть основных категорий:
- Строительные материалы (цемент, кирпич, песок)
- Инструменты (электроинструменты, ручной инструмент)
- Отделочные материалы (краска, обои, плитка, ламинат)
- Электрика (провода, розетки, светильники)
- Сантехника (трубы, фитинги, смесители)
- Окна и двери

### Фильтрация

Товары можно фильтровать по трем основным категориям: материалы, инструменты и отделка. Также реализован поиск по названию и описанию товара.

## Функциональность

- Фильтрация товаров по категориям
- Поиск товаров по запросу
- Добавление товаров в корзину с визуальной обратной связью
- Анимация появления элементов при прокрутке
- Адаптивное мобильное меню
- Плавная прокрутка между секциями
- Форма обратной связи

## Особенности для строительного магазина

В отличие от обычных интернет-магазинов, строительный магазин имеет свои особенности:

- Большие объемы товаров (тонны, кубометры)
- Необходимость расчета количества материалов
- Разные единицы измерения
- Важность доставки крупногабаритных товаров
- Консультации специалистов по подбору материалов

Эти особенности можно учесть при дальнейшей разработке проекта.
